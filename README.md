# Manual QA Testing Project – SauceDemo

## Application Tested
https://www.saucedemo.com

## Project Type
Manual Functional Testing

## Objective
The objective of this project is to perform manual testing on the SauceDemo ecommerce website to verify core functionalities such as login, product listing, cart management, checkout, and logout.

## Testing Scope
The following modules were tested:

- Login functionality
- Product page
- Add to cart
- Remove item from cart
- Checkout process
- Product sorting
- Logout functionality

## Testing Types Performed
- Functional Testing
- Regression Testing
- Smoke Testing

## Tools Used
- Manual Testing
- Microsoft Excel for Test Case Documentation
- GitHub for Project Portfolio

## Test Environment
Browser: Google Chrome
Operating System: Windows 11
Application URL: https://www.saucedemo.com


## Test Artifacts Included
This repository contains the following testing artifacts:

- Test Cases
- Bug Reports
- Test Execution Report

## 🐞 Bug Tracking (Jira)

Bugs were reported and tracked using Jira.

## Test Execution Summary

Total Test Cases: 13  
Passed: 11  
Failed: 2

## Bugs Identified

1. Checkout page loads even when the cart is empty
2. Reset state does not remove items from cart
